Turing-complete system.

There are many possible algorithms. Mathematician Alan Turing showed that everything that can be calculated (and some things can not) can be done by Turing machine. For computational system system to be Turing-complete, following requirements should be met:
  1. It should provide branching
  2. It should provide loops.
  3. Memory should be infinite.

Branching (conditional execution) in LMC

How branching can be achieved?
You can recall that algorithm is a sequence of operations and each operation is stored in memory at certain address. Program Counter (PC, pointer to memory location) keeps track of what is done.
If we want to interrupt sequential execution, simple way is to change the value stored in PC.

Unconditional jump - BRA XX.

First and not really conditional operator is BRA XX. It writes value XX into Program Counter.
Here is an example of unconditional jump:
INP
STO 20
INP
STO 21
BRA 0
HLT
Programs goes around forever - we made infinite loop.

Jump if 0 - BRZ XX

Here is true conditional jump: value XX is written into PC if Accumulator contains 0. Overall result is conditional jump.
Here is the example - program which count down from specified number down to 0. 

INP
STO 20
INP
STO 21
SUB 20
OUT
BRZ 8
BRA 4
HLT
